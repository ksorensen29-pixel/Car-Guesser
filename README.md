# Car-Guesser
Common Questions:

What coding luanguages are in use in this repo?
JS, HTML, and CSS
What does this project do?
It will act like a guessing game for car fans to guess on info on a rotating selection of cars.
Is AI used in this project?
as amateur developers we sometimes need to lean on it, but despite that we are attempting to make this project mostly human generated.
What problem are you solving?
Often fans and enthusiasts of many different subjects and hobbies struggle to find ways to quickly and enjoyable learn and memorize information on their favorite subjects, we hope that our game will allow car fans to not only find new cars, but also learn facts and info on them.
